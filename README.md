# Summit Landscape Boilerplate

This is a landscaping business website boilerplate built with [Next.js](https://nextjs.org) and React.

## Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- ESLint

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize for Your Business

- Update branding and contact details in `app/page.tsx`
- Update SEO metadata in `app/layout.tsx`
- Adjust colors and global styles in `app/globals.css`

## Production Build

```bash
npm run build
npm start
```
